# Privacy

One Step stores all goal data locally on your Mac. Nothing leaves your device.

- **No account required.** The app launches and works without sign-up.
- **No server uploads.** Goal titles, daily actions, completion records, and all other user data stay in the local App Group container.
- **No analytics or tracking.** The app does not collect usage data, crash reports, or behavioral metrics.
- **Shared storage is local.** The main app and desktop Widget read and write the same local data through the macOS App Group container (`group.dev.dynnfa.OneStep`). This container lives on your disk, inside your sandbox.

## Future Features

If One Step adds sync, export, or any feature that moves data off the local device, this document will be updated before that feature ships to describe what data moves, where it goes, and what control the user has.
